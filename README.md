Nirma SEE Calculator - Full App
--------------------------------
This project includes:
- Firebase v9 auth (Google Sign-In)
- Firestore to store student attempts
- SEE calculator replicating Excel logic
- Admin panel (admin email: dhruvbhalala07@gmail.com)
- PDF export (jsPDF)
- Reset marks button
- Ready to deploy to Render/Vercel

Setup:
1. Install Node.js and npm.
2. Run `npm install` in project root.
3. Update `public/firebase.js` with your firebaseConfig if different.
4. Run `npm start` and open http://localhost:3000/login

Admin:
Login with dhruvbhalala07@gmail.com to be redirected to /admin.
